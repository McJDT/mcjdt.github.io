# Getting Started

This is your first guide! You can edit this file directly.

## What to Do

1. Edit the `.md` files in the `content/` folder
2. Add new pages to `manifest.json`
3. Refresh the browser to see your changes

## Writing in Markdown

Markdown is simple and readable. Here's what you can do:

### Headings

Use `#` for headings:
- `# Heading 1`
- `## Heading 2`
- `### Heading 3`

### Lists

- Item 1
- Item 2
- Item 3

### Code

Use backticks for inline `code` or triple backticks for code blocks:

```javascript
console.log('Hello, world!');
```
