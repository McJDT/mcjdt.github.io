# Writing Guide

Tips and tricks for writing great documentation.

## Keep It Simple

- Use clear language
- Short paragraphs
- Bullet points for lists

## Structure Your Content

- Start with a clear title
- Use headings to break up sections
- Add examples when helpful

## Adding Images

Place images in the `images/` folder, then reference them:

```markdown
![Image description](images/filename.jpg)
```

The image will display inline in your documentation.

## Code Examples

When sharing code, use triple backticks with the language:

```python
def hello():
    print("Hello, world!")
```

This makes your documentation more readable and professional!
