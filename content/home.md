# Welcome to My Docs

This is a markdown-based documentation site. You can edit all content by simply modifying the `.md` files in the `content/` folder.

## How It Works

- **Content**: All pages are written in Markdown (`.md` files) in the `content/` folder
- **Images**: Store images in the `images/` folder and reference them in your markdown
- **Organization**: Create subfolders to organize your content (like `guides/`, `tutorials/`, etc.)
- **Navigation**: The sidebar is automatically generated from `manifest.json`

## Adding Content

1. Create a new `.md` file in the `content/` folder (or a subfolder)
2. Add an entry to `manifest.json` to include it in the sidebar
3. Refresh the page - your new content appears!

## Example Markdown

You can use standard markdown syntax:

- **Bold text**: `**bold**`
- *Italic text*: `*italic*`
- [Links](https://example.com)
- Lists and more

## Images

Store images in the `images/` folder and reference them like this:

```
![My Image](images/my-photo.jpg)
```

Happy writing!
